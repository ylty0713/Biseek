# 🚀 AI Trading Assistant

An intelligent, modular AI-powered trading assistant that combines **natural language interaction**, **market analysis**, and **real-time monitoring** for cryptocurrency markets.

---

## ✨ Features

- 🤖 **Conversational AI**
  - Chat naturally with the assistant
  - Automatic intent detection (chat vs trading vs alert)

- 📊 **Multi-Timeframe Market Analysis**
  - Supports 1h / 4h / 1d analysis
  - Trend structure + support/resistance

- 📈 **Technical Indicators**
  - EMA, RSI, MACD
  - Custom indicator extensions supported

- 💰 **Market Sentiment & Funding Data**
  - Funding rates
  - Open interest (OI)
  - Market positioning insights

- 💥 **Liquidation Analysis**
  - Detect liquidation clusters
  - Identify potential stop-hunt zones

- 🔔 **Smart Price Alerts**
  - Natural language alerts:
    - “BTC above 70000”
    - “ETH below 2000”
  - Multi-threaded real-time monitoring

- 🧠 **Intent Recognition Engine**
  - Automatically routes:
    - Chat → LLM
    - Analysis → Data + AI
    - Alerts → Execution system

- 🔄 **Multi-Model Support**
  - OpenAI
  - DeepSeek
  - Easily extendable

---
